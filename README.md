# Calculadora em Assembly
<img src="CalculadoraAssembly.PNG">

## Sobre
<p>O projeto consite em um gerador de código para uma calculadora em Assembly que realiza as funções de soma, subtração, multiplicação, divisão, potenciação, raiz quadrada, fatorial e fibonacci.</p>
<p>No arquivo Documentação.pdf é possível encontrar maiores informações sobre a execução do projeto.</p>
